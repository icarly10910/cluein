Cluein Tutorial
===============
1. Start cluein by clone the cluein repo on your local environment. 
2. After you download all your files, make sure you have downloaded the newest version of Django and Python3.
3. After you have downloaded both of these, you can open up the repo on any text editor you prefer.
4. After configuring the repo with Django, you can run 'python3 manage.py runserver' to start up my web application.
5. You can now start adding articles of clothing you like!
